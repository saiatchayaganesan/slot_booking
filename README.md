# Slot Booking BTS

A **Flutter app** for booking slots for BTS events or concerts. This app allows users to view events, select available time slots, and confirm their bookings.

---

## **Features**

- View a list of BTS events.
- Check available slots for each event.
- Book a slot and confirm the booking.
- Easy-to-use and responsive UI.



- Flutter SDK installed
- Android Studio with Flutter plugin
- Device or emulator to run the app

Future Enhancements:

Automatic APK download and update system.

User authentication for booking history.

Push notifications for new events or slot availability.

Improved UI/UX with animations.
